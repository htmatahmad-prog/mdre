# حل مشكلة ظهور النص العربي بشكل مقلوب

## المشكلة
تحدث مشكلة انقلاب النص العربي عادة بسبب:
1. **عدم تطابق الترميز**: النظام لا يستخدم UTF-8 بشكل صحيح
2. **عدم دعم النصوص ثنائية الاتجاه**: النظام لا يتعرف على أن العربية تُكتب من اليمين إلى اليسار (RTL)
3. **مشاكل في إعدادات الترمينال**: الترمينال غير مُعدّ لعرض النصوص العربية

## الحلول

### 1. إعداد النظام لاستخدام UTF-8
```bash
export LC_ALL=en_US.UTF-8
export LANG=en_US.UTF-8
```

### 2. تشغيل بايثون بترميز UTF-8
```bash
export PYTHONIOENCODING=utf-8
python3 your_script.py
```

### 3. التأكد من أن ملف بايثون محفوظ بترميز UTF-8
يجب أن يحتوي الملف على:
```python
# -*- coding: utf-8 -*-
```

### 4. للتطبيقات الويب (HTML/CSS)
أضف هذه الأسطر في الـ `<head>`:
```html
<meta charset="UTF-8">
```

أو في CSS:
```css
html {
    direction: rtl;
}
```

### 5. للتطبيقات التي تستخدم PyQt أو Tkinter
```python
# For PyQt
app = QApplication([])
app.setProperty("ansi_utf8", True)

# For Tkinter
root = Tk()
root.option_add("*Font", "Arial Unicode MS")
```
